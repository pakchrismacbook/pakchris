Tools

The Pig Game

A small app based on a die rolling game of the same name. Players will take turns rolling a die, first to 50 wins. This app was created using HTML, CSS, and vanilla Javascript. No libraries or boilerplate code.

This app is also responsive to small screen resolutions so it can be played on mobile devices.
